## *class* algokit_utils.network_clients.AlgoClientConfig

AlgoClientConfig

Connection details for connecting to an [`algosdk.v2client.algod.AlgodClient`](../algosdk/algosdk.v2client.algod.mdx#algosdk.v2client.algod.AlgodClient) or
[`algosdk.v2client.indexer.IndexerClient`](../algosdk/algosdk.v2client.indexer.mdx#algosdk.v2client.indexer.IndexerClient)

<a id="algokit_utils.network_clients.AlgoClientConfig.server"></a>

### server

server *: str*

None

URL for the service e.g. `http://localhost:4001` or `https://testnet-api.algonode.cloud`

<a id="algokit_utils.network_clients.AlgoClientConfig.token"></a>

### token

token *: str*

None

API Token to authenticate with the service

## algokit_utils.network_clients.get_kmd_client_from_algod_client
