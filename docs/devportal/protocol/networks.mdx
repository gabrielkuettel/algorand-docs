---
title: Networks
description: Information about Algorand's public networks
---

Algorand has three public networks: MainNet, TestNet, and BetaNet. This section provides details about each of these networks that will help you validate the integrity of your connection to them.

Each network reference, contains the following information:

<table>
  <tr>
    <th>Version</th>
    <td>
      The latest protocol software version. Should match goal -v or GET /versions build version.
    </td>
  </tr>
  <tr>
    <th>Release Version</th>
    <td>A link to the official release notes where you can view all the latest changes.</td>
  </tr>
  <tr>
    <th>Genesis ID</th>
    <td>
      A human-readable identifier for the network. This should not be used as a unique identifier.
    </td>
  </tr>
  <tr>
    <th>Genesis Hash</th>
    <td>
      The unique identifier for the network, present in every transaction. Validate that your
      transactions match the network you plan to submit them to.
    </td>
  </tr>
  <tr>
    <th>FeeSink Address</th>
    <td>
      Where all fees from transactions are sent. The FeeSink can only spend to the RewardsPool
      account.
    </td>
  </tr>
  <tr>
    <th>RewardsPool Address</th>
    <td>
      Originally used to distribute rewards to balance-holding accounts. Currently this account is
      not used.
    </td>
  </tr>
  <tr>
    <th>Faucet</th>
    <td>Link to a faucet (TestNet and BetaNet only)</td>
  </tr>
</table>
